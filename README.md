GeoLocalizacao
📝 Descrição
O GeoLocalizacao é uma aplicação mobile desenvolvida com React Native e Expo (SDK 54) que tem como objetivo explorar e implementar funcionalidades de geolocalização em tempo real. O projeto foca na integração com APIs de mapas e sensores de GPS para fornecer coordenadas precisas, servindo como base para soluções de logística, rastreamento e serviços baseados em localização.
🚀 Tecnologias Utilizadas
Frontend: React Native (v0.81.5)
Framework: Expo (v54.0.33)
Linguagem: JavaScript (ES6+)
Ambiente de Execução: Node.js
🛠️ Funcionalidades Planejadas
Solicitação de permissões de GPS (Android/iOS).
Captura de latitude e longitude em tempo real.
Renderização de mapas interativos.
Integração com serviços de Geocoding (converter coordenadas em endereços).
⚙️ Como Rodar o Projeto
Pré-requisitos
Node.js instalado.
App Expo Go instalado no smartphone ou um emulador configurado.
Passos
Clone o repositório:

git clone <url-do-repositorio>

Instale as dependências:

npm install

Inicie o servidor de desenvolvimento:

npm start

Acesse o App: Escaneie o QR Code exibido no terminal com a câmera do seu celular (iOS) ou pelo app Expo Go (Android).
